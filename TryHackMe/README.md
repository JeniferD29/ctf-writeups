# TryHackMe Writeups

## About TryHackMe

TryHackMe is an interactive cybersecurity learning platform that provides hands-on labs and guided learning paths across a wide range of cybersecurity domains. It enables learners to develop practical skills through real-world scenarios, Capture The Flag (CTF) challenges, and virtual environments.

The platform covers topics including:

- Penetration Testing
- Web Application Security
- Network Security
- Active Directory
- Digital Forensics
- Threat Intelligence
- Security Operations (SOC)
- Wireless Security
- Linux & Windows Privilege Escalation
- Cloud Security

## About This Directory

This directory contains my personal walkthroughs and learning notes for the TryHackMe rooms I have completed. Each room is documented with the methodology, commands used, key concepts learned, and screenshots where applicable.

The walkthroughs are intended for educational purposes and to document my continuous learning journey in cybersecurity.

> **Note:** All exercises were completed in authorized TryHackMe lab environments.
